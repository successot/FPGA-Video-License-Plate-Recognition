# License Plate Detection and Recognition

本目录包含车牌检测与车牌字符识别相关的训练、评估和边缘部署代码，可作为 FPGA 视频采集链路后端的 AI 识别模块使用。

## 功能概述

- 车牌检测模型训练与测试
- 车牌区域裁剪与数据预处理
- 基于 CTC 的车牌字符识别训练
- 多阶段 OCR 训练与微调流程
- RK3568 平台部署程序与模型文件组织

## 目录说明

```text
.
├── detector_train/       # 车牌检测训练与测试代码
├── ocr_train/            # OCR 识别训练、微调和数据处理代码
├── weights/              # 模型权重，建议使用 Git LFS 或 Release 管理
└── rk3568_app/           # RK3568 平台部署相关文件
```

> 实际目录名称以仓库中的文件为准。若模型权重较大，建议不要直接提交到普通 Git 历史中，可改用 Git LFS 或 GitHub Releases。

## 使用建议

1. 准备车牌检测和识别数据集。
2. 根据训练脚本中的参数配置数据路径、字符集路径和输出路径。
3. 先训练检测模型，再基于检测结果裁剪车牌区域。
4. 使用 OCR 训练脚本完成字符识别模型训练。
5. 将训练后的模型部署到边缘平台或与 FPGA 视频链路集成。

## 注意事项

- 本仓库已移除缓存、日志、临时文件和本机绝对路径。
- 数据集未随项目提供，请按照脚本参数自行准备。
- 训练权重和可执行文件如体积较大，建议通过 Release 或 Git LFS 管理。
- 部署环境可能需要根据目标平台重新配置运行库和模型路径。
